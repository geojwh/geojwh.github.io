Adelbert's Brewery Web Map

Web map created for Adelbert's Brewery in Austin, TX. Utilized the extended Mapbox.js JavaScript library (based on Leaflet) and the Leaflet Markercluster Plugin. The map displays locations where consumers can purchase Adelbert's beer. The map is live on the brewery's website.

DPS AGOL Web AppBuilder

Map application created using the ArcGIS Online Web AppBuilder for the Texas DPS Highway Safety Operations Center (HSOC). The map is specifically for the Highway Safety Assessment weekly product created by the research group. The application is under review and not yet live.

DPS Leaflet Map

Web map created as a potential open-source alternative to the Texas DPS HSOC ArcGIS Online map. This map uses the Leaflet.js library to display weekly fatal crashes for the Highway Safety Assessment, using geojson for district boundaries and the markers. Currently, the HSOC is in a testing phase to determine the best method to deploy a web map to allow interactive investigation of crash events versus static map views. ***This map is under construction.
