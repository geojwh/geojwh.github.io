crash fatality web map - summer 2015

a web map created using arcgis online web appbuilder to display weekly traffic crash fatalities across the state of texas. data acquired from the txdot crash records information system (c.r.i.s.).


adelbert's brewery web map - summer 2015

created for an austin, tx craft brewery named adelbert's using the extended mapbox javascript library, as well as the leaflet marker cluster plugin, to show nationwide locations where adelbert's beer can be purchased.
