projects

fall 2015

  • web map created as an open source alternative to the tx dps hsoc arcgis online map which uses the leaflet.js library to display fatal crashes for the highway safety assessment. currently the group is researching the best method for deploying a web map to allow interactive investigation of crash events versus static map views. map

summer 2015

  • web map created for adelbert's brewery in austin, tx. utilized the extended mapbox.js library (based on leaflet) and the leaflet markercluster plugin. map displays locations where consumers can purchase the brewery's beer. the map is live on the brewery's website. map

  • **updated 12/30/15** map application created using the arcgis online web appbuilder for the texas dps highway safety operations center (hsoc) highway safety assessment (hsa). the application is under review and not yet live. map
