projects

  • **updated 01/13/16** map application created using the arcgis online web appbuilder for the texas dps highway safety operations center (hsoc) highway safety assessment (hsa). the application is under review and not yet live. VIEW WEB MAP

  • web map created as an open source alternative to the tx dps hsoc arcgis online map. this map uses the leaflet.js library to display fatal crashes for the highway safety assessment, as well as geojson for the state and country boundaries. currently the group is researching the best method for deploying a web map to allow interactive investigation of crash events versus static map views. VIEW WEB MAP

  • web map created for adelbert's brewery in austin, tx. utilized the extended mapbox.js library (based on leaflet) and the leaflet markercluster plugin. map displays locations where consumers can purchase the brewery's beer. the map is live on the brewery's website.VIEW WEB MAP
