projects

  • web map created using the leaflet javascript library to display fatal crashes for the texas highway safety operations center's (hsoc) highway safety assessment (hsa) weekly product. currently the group is researching the best method for deploying a web map to allow interactive investigation of crash events versus static map views. map

  • map appication created using the arcgis online web appbuilder for the hsoc's hsa weekly product. the application is completed but has not been adopted yet due to investigation into lighter/quicker web map alternatives. map

  • web map created for adelbert's brewery in austin, tx. utilized the extended mapbox javascript library (based on leaflet) and the leaflet marker cluster plugin. map displays locations where consumers can purchase the brewery's beer. the map is live on the brewery's website. map
